## 🔍 ドキュメントの探し方

### 開発を始める方

1. [クイックスタート](./getting-started/quick-start.md) を読む
2. [開発環境セットアップ](./getting-started/setup.md) で環境を構築
3. [設計ガイド](./architecture/design-guide.md) でアーキテクチャを理解

### コードを理解したい方

1. [ディレクトリ構造](./architecture/directory-structure.md) で構造を把握
2. [設計ガイド](./architecture/design-guide.md) で設計思想を理解
3. [API リファレンス](./reference/api.md) で詳細を確認

### デプロイしたい方

1. [デプロイメントガイド](./guides/deployment.md) を参照
2. [トラブルシューティング](./guides/troubleshooting.md) で問題を解決

### 貢献したい方

1. [貢献ガイドライン](./guides/contributing.md) を読む
2. [セキュリティポリシー](./guides/security.md) を確認
